# 例句库

按类型分类的精选文风示例，来自 horse 2007-2026 年的 Twitter Highlights。

## 使用方式

写作时如果对某个节奏或结构不确信，可以到这里对照原帖找感觉。

## 分类

- `短判断` — 一句或几句话的犀利判断
- `结构化推理` — 从现象到结构的渐深层析
- `生活记录` — 从城市/咖啡/书开始的具体写作
- `平台与技术批评` — 对互联网/产品/媒体的分析
- `自我评价` — 装那啥结束式的自我拆台
- `批评与愤怒` — 有结构的愤怒，不纯骂

## 来源

全部素材来自 horse 在 Twitter/X 上的公开写作（2007-2026）。
这些帖子已被精选为高价值/代表性样本。
