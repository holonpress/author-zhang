# 文风质量审查表

逐条检查你的文章。每条用 Yes / No / Partial 标记。

---

## 一、思维运动检查（核心）

### M1. 触发物
文章是否从一件具体的事情开始？不是从抽象观点开始的。

- ✅ 好的：一条新闻标题、一句朋友说的话、一个咖啡馆的细节、一个平台上的遭遇
- ❌ 不好的：直接开始讲道理、引用统计数据开头
- 🔧 不通过 → 重写第一段，找个具体的触发物。参考 `skills/horse-style-writer/references/examples/结构化推理.md` 和 `skills/horse-style-writer/references/examples/生活记录.md`

### M2. 有一个真正的判断
文章有一个自己的判断——即使是带着 `我觉得` `大概` `可能` 的。

- ✅ 好的：判断很明确，即使带着犹疑
- ❌ 不好的：纯描述、中立汇总、态度模糊
- 🔧 不通过 → 问自己：这篇文章里，我到底在说"我觉得这是什么问题"？

### M3. 反框架
拒绝了第一层解释。推动了"大家以为的问题是X，但其实是Y"。

- 检查：有没有 `不是A，而是B` / `看上去是A，其实更像B` / `问题不在X，而在Y`
- 🔧 不通过 → 对照 `skills/horse-style-writer/references/style-fingerprint.md`「论证结构」节，看示例中如何找到反向框架

### M4. 语境化
不只是说"这不对"，而是解释了它为什么这样运行。

- ✅ 好的：有历史脉络、结构分析、机制拆解
- ❌ 不好的：只停留在当下事件的表层
- 🔧 不通过 → 问自己：这个问题放到更长的历史里看是什么样子？背后有什么机制在运转？参考 `skills/horse-style-writer/references/examples/结构化推理.md`

### M5. 回到人
分析最后落到了具体的人身上。

- 检查：文章中是否有具体的"人"出现——用户、朋友、某个群体、受伤的人、普通人
- 🔧 不通过 → 问自己：这个分析如果不落地，对谁都没有意义。找个具体的人或处境来收束。参考 `skills/horse-style-writer/references/style-fingerprint.md`「情绪结构」

### M6. 结尾有余味
结尾不是金句式总结，而是留下持续压力。两个有效类型：

**类型 A — 悬而未决**：结尾停在未完成的场景、悬着的判断、不确定的未来。
- ✅ "废墟之间有一根脆弱的线……它可能仍然在那里，不松不紧地绷着"

**类型 B — 自反一击**：结尾让文章的框架反过来击中自身。
- ✅ "这段文字本身也困在它批判的结构里"

- ❌ 不好的："所以我们应该…"、"综上所述…"、"这就是生活的真谛…"
- 🔧 不通过 → 删掉最后一段总结性的话。问自己：这个结尾是让读者合上页面后还在想，还是把答案喂给读者了？

## 二、修辞检查

### R1. `不是A而是B` 使用频率
- 1-3 次：✅
- 4 次以上 → 替换其中一两个
- 0 次 → 检查是否有一个地方可以用它来推动分析

### R2. `我觉得` / `大概` / `可能` 犹疑词
- 2-5 次：✅
- 0 次 → 可能写得太肯定了
- 8 次以上 → 考虑加强判断力度

### R3. 编号拆解
- 如果是真正的结构化需要：✅
- 如果只是为了"看起来有条理" → 去掉或合并

### R4. 括号注释
- 每段 ≤ 2 处：✅
- 一页 > 3 处 → 削减

## 三、文本质素检查（新增 — 品控升级）

### V1. 句子节奏
检查句子长度模式是否单一，有无通过短句制造击打感。

- ✅ 连续 3 句以上有不同长度模式
- ✅ 至少有一处用短句（≤ 10 字）收束一个段落或转折
- ❌ 连续 4 句以上主谓宾结构不变
- ❌ 每段连续多个句子都以同样的词开头（"但" "不过" "所以" "问题在于"）
- 🔧 有问题 → 打断节奏：在读起来最平坦的地方插一个短句，或者合并两个相邻的短句为一个长复合句

### V2. 段落弧线
每段应有自己的微弧线（进入→推进→转折/着落），而不是在同一个平面上平铺信息。

- ✅ 每段读起来有方向感和推进感
- ❌ 一段只做一件事的清单式展开（"首先…其次…最后…"）
- ❌ 段落在同一个论证层面徘徊，没有深化或转折
- 🔧 不通过 → 每段写完问自己：这一段从哪开始、到哪结束？中间有没有一个让读者觉得"噢"的时刻？如果没有，这段可能是信息堆砌，需要重新组织

### V3. 具体性审查
抽象概念后面是否有具体名词或场景支撑？

- ✅ 抽象判断后有具体例子、场景、或名词
- ✅ "人"是具体的（用户、朋友、某地居民、某个职业）——不是泛指的"人民""大众""我们"
- ❌ "日常生活的温暖" "社会的复杂性" "历史的必然" 等空泛抽象
- ❌ 连续使用两个以上抽象名词串（结构+机制+语境+权力一起出现而不落地）
- 🔧 不通过 → 每发现一个抽象词，问自己：能不能换成具体名词？能不能举一个例子？如果不换读者会走神吗？

### V4. 衔接过渡
段与段之间是滑过去的还是跳过去的？

- ✅ 段间有明显推进感，下一段是对上一段的深化、转折、或具体化
- ✅ 过渡方式多样化：除了 "但是" 之外有 "回到…" "有意思的是…" "这里有一个细节" "反过来看" 等
- ❌ 每段结尾是句号结束、没有向下一段递进的拉力
- ❌ "另一方面…与此同时…" 式平行移动（段与段之间没有逻辑推进，只有话题切换）
- 🔧 不通过 → 在两段之间加一句过渡句，明确下一段跟上一段的关系：是举例？转折？深化？反面？

## 四、边界检查

### B1. 广告/自媒体语气
- ❌ "必看" "震惊" "建议收藏" "作为一个XX，我…"
- 🔧 有 → 改写成平实的语气

### B2. 过度文艺化
- ❌ "风很轻，云很淡"式抒情、堆砌修辞
- 🔧 有 → 删掉多余的形容词和抒情句

### B3. 确定性幻觉
- ❌ "这才是真相" "事实很清楚" "所有人都知道"
- 🔧 有 → 加限定词：`可能有` `我倾向于认为` `至少现阶段看来`

### B4. 编造
- ❌ 虚构的引语、数据、个人经历、朋友的话、来源不明的"据专家称"
- 🔧 有 → 删除。诚实标注"我记得"或"印象中"

## 五、总评

| 维度 | 项目 | 评分 | 不通过时的修复 |
|------|------|------|---------------|
| 思维运动 | 触发物 (M1) | | 重写第一段 |
| | 真判断 (M2) | | 明确立场 |
| | 反框架 (M3) | | 对照指纹「论证结构」 |
| | 语境化 (M4) | | 加历史/结构/机制层 |
| | 回到人 (M5) | | 找个具体人收束 |
| | 余味结尾 (M6) | | 删总结，区分结尾类型 |
| 修辞节制 | R1-R4 | | 对照频率限制调整 |
| 文本质素 | 句子节奏 (V1) | | 打断单一节奏，插短句 |
| | 段落弧线 (V2) | | 确认每段有推进感 |
| | 具体性 (V3) | | 抽象词→具体名词/场景 |
| | 衔接过渡 (V4) | | 加过渡句，明确段间关系 |
| 边界合规 | B1-B4 | | 删除越界内容 |

**通过标准：**
- M1-M6 ≥ 4 Yes
- V1-V4 ≥ 3 Yes（新增文本质素要求）
- 无 B1-B4 边界违规
- 修辞 R1-R4 不超标

**关于 Partial：** Partial 视为 No，不计入通过计数。

**否则：** 修改后重新审查（回 Phase 4，不直接改）。
