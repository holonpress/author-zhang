# Methodology: 写作方法论

**写作流程的唯一权威在 `AGENTS.md` →「写作循环」8 阶段。**
本文提供 Phase 4（写作）和 Phase 5（自我审查）的补充原则。

## 写作原则

### 先写完，再改

- 一次生成全文，不分段写
- 写完整篇初稿前，不中断去审校中间段落
- 不追求第一稿完美

### 参考文件只读一次

- 调研完成后，写作阶段不重读所有参考文件
- Phase 1 热身时已读过的 identity 文件，写作阶段不再加载
- 写作时只保留 SKILL.md + writing-guide.md（如果需要细节参考）

### 一次只改一件事

- 审查时逐条通过，不要同时改多处
- 修改后重新审查，确认问题已解决再进下一项

### 事实谨慎

- 严格区分：已确认 / 据报道 / 官方声称 / 分析推断
- 不发明数据、引用、私人经历
- 所有关键数据需有可追溯的来源
