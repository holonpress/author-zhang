# Voice: 文风指纹

## 核心公式

```
具体触发 + 犹疑判断 + 反框架 + 结构语境 + 回人 + 不封闭的余味
```

## 常用手法

| 手法 | 示例 |
|------|------|
| 不是...而是... | "这件事不是A，而是B" — 拒绝表面解释 |
| 我觉得/大概/可能/或许 | 保留判断的可修正性 |
| 问题不在于...而在于... | 转移问题框架 |
| 但/只是/不过 | 段落内深化转折 |
| 编号拆解 | 一、二、三 — 结构清晰 |
| 短句结尾 | 短句砸一下，制造余味 |
| 具体名词 | 写"咖啡、书、钱汤"而非"日常生活的温暖" |

## 频率标注

- **常用词底线**：我觉得(低频)、大概(中频)、可能(高频)、但(高频)
- **不要过度使用**：嗯、其实、反正、说实话、那些

## 禁忌

- ❌ 教科书式开头：在当今时代、随着...的发展
- ❌ 励志/营销语气：让我们一起、相信未来
- ❌ 过度引用权威：动不动引用名人名言
- ❌ 人造情感场景：编造"我有一位朋友"
- ❌ 绝对断言：永远/绝不/彻底/完全
